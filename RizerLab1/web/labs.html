My experience with front-end frameworks and languages is fairly limited. The advanced CSS techniques I used for this lab include the "border-radius" attribute, which allows for rounded edges in my content div and avoids a blocky, unappealing style, as well as the "width" attribute applied to the banjo picture where I can resize the page and maintain a proper ration between the text and the image.

In this lab, I learned about referencing code snippets with the Angular-like "ng" tags, which were very useful for the common elements of my web pages such as nav bars and footers. I also learned how powerful CSS is with respect to formatting a page, which until this class I just assumed made webpages slightly prettier, and the bulk of formatting and positioning on a web page was predominantly HTML.
